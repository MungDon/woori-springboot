반복문

th:each="str : ${list}"

forEach문 구조,

"변수 : 반복대상" => 기본구조

th:each="str, loop:${list}"

loop 는 반복문의 속성을 가진 변수

loop 의 이름은 사용자지정이다

반복문의 속성

loop.index : loop 의 순서(반복순서 , 0부터 1씩 증가)

loop.count : loop 의 순서(반복순서 , 1부터 1씩 증가)

loop.size : 반복하는 객체의 요소 개수(ex)list 의 요소개수)

loop.first : loop 의 첫번째 순서인 경우 true

loop.last : loop 의 마지막 순서인 경우 true

loop.odd : loop 의 홀수번째 순서인 경우 true

loop.even : loop 의 짝수번째 순서인 경우 true

loop.current : 현재 대입된 객체
